A CLEAN WEBSITE TEMPLATE, THAT LOOKS DAMN GOOD!

MIA is a premium one-page HTML5/CSS3 website template, based on Twitter Bootstrap 3.2.0 framework. MIA is designed with agencies and small teams in mind and can help them to launch a simple, modern and professional website in no time! It is really easy to use and configure and includes full documentation.

WHAT MIA CAN GIVE TO YOUR BRAND?

Clean Design & Code

Semantic, well indented code. Full documentation. Clean file structure. We believe you will find your way there in no time!

ease of use

MIA is featuring easy separation and re-arrangement of each content section, making the HTML code of this template, very easy to edit or customize.

Robust Foundation

MIA is based on Twitter Bootstrap framework, giving your next website, a rock-solid, cross-browser and responsive foundation that is used by many designers out there.
